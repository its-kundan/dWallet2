## üõ†Ô∏è Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/its-kundan/energi-frontend.git
   ```
2. Install dependencies:
   ```bash
   cd energi-frontend
   yarn install
   ```
3. Start the development server:
   ```bash
   yarn dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
